🧱 Cracked Brick Components

New Components Added

I've created two new components for you:

  • CrackedBrick - Individual cracked brick primitive with different crack intensities
  • CrackedDestructibleWall - Destructible wall made of cracked bricks with breakable system integration

1. CrackedBrick Component

A primitive component for individual cracked bricks with three crack intensity levels:

import { CrackedBrick } from './components/primitives/elements';

// Light cracks
<CrackedBrick
  position={[0, 0, 0]}
  crackIntensity="light"
  size={[1, 0.5, 0.5]}
  enabled={true}
  breakingOptions={{
    fragmentCount: 6,
    fractureImpulse: 0.8,
    minSizeForFracture: 0.2,
    maxSizeForFracture: 0.6,
  }}
  onBreak={(impactPoint) => console.log('Brick broken!')}
/>

// Medium cracks
<CrackedBrick
  position={[2, 0, 0]}
  crackIntensity="medium"
  size={[1, 0.5, 0.5]}
  enabled={true}
  breakingOptions={{
    fragmentCount: 8,
    fractureImpulse: 0.8,
    minSizeForFracture: 0.2,
    maxSizeForFracture: 0.6,
  }}
  onBreak={(impactPoint) => console.log('Brick broken!')}
/>

// Heavy cracks
<CrackedBrick
  position={[4, 0, 0]}
  crackIntensity="heavy"
  size={[1, 0.5, 0.5]}
  enabled={true}
  breakingOptions={{
    fragmentCount: 10,
    fractureImpulse: 0.8,
    minSizeForFracture: 0.2,
    maxSizeForFracture: 0.6,
  }}
  onBreak={(impactPoint) => console.log('Brick broken!')}
/>

Features:

  • Light cracks: 2 crack lines, minimal damage
  • Medium cracks: 4 crack lines, moderate damage
  • Heavy cracks: 6 crack lines, chipped edges, moss patches, dust particles
  • Uses brick texture from the texture system
  • Integrated with breakable system
  • Physics-based breaking with fragments

2. CrackedDestructibleWall Component

A destructible wall made of cracked bricks that integrates with the breakable system:

import { CrackedDestructibleWall } from './components/primitives/objects';

<CrackedDestructibleWall
  position={[0, 1.5, 0]}
  rotation={[0, 0, 0]}
  health={3}
  bombRequired={false}
  enabled={true}
  breakingOptions={{
    fragmentCount: 12,
    fractureImpulse: 1.0,
    minSizeForFracture: 0.3,
    maxSizeForFracture: 0.8,
  }}
  onBreak={(impactPoint) => console.log('Wall damaged!')}
  onDestroy={() => console.log('Wall destroyed!')}
/>

Features:

  • Wall made of individual cracked bricks with mortar lines
  • Health system with visual damage progression
  • Crack intensity increases as health decreases
  • Optional bomb requirement
  • Integrated with breakable system for physics-based destruction
  • Visual health bar and interaction prompts
  • Damage particles and visual effects

4. Integration with Existing System

The components are fully integrated with your existing systems:

  • Uses withOptionalBreaking HOC for breakable functionality
  • Compatible with the texture loading system
  • Uses Rapier physics for collision detection
  • Follows the same prop patterns as other primitives
  • Exported through the main primitives index

5. Usage in Your App

To use these components in your application:

// Import the components
import { CrackedBrick, CrackedDestructibleWall } from './components/primitives';

// Or import from specific modules
import { CrackedBrick } from './components/primitives/elements';
import { CrackedDestructibleWall } from './components/primitives/objects';

// Use in your 3D scene
function MyScene() {
  return (
    <group>
      {/* Individual cracked bricks */}
      <CrackedBrick
        position={[0, 0, 0]}
        crackIntensity="medium"
        enabled={true}
        onBreak={(point) => console.log('Brick broken!')}
      />
      
      {/* Destructible wall */}
      <CrackedDestructibleWall
        position={[5, 1.5, 0]}
        health={3}
        enabled={true}
        onDestroy={() => console.log('Wall destroyed!')}
      />
    </group>
  );
}
